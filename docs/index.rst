Welcome to AWS Security Group Mapper's documentation!
==================================================

AWS Security Group Mapper is a Python tool for visualizing AWS security group relationships and generating interactive graphs. This documentation provides detailed information about the tool's usage, architecture, and API reference.

Getting Started
--------------

Check out the :doc:`installation` guide to install the package, then move on to the :doc:`usage` section to learn how to use AWS Security Group Mapper.

Features
--------

- Security Group Mapping
    - Comprehensive mapping within and across VPCs
    - Support for multi-region analysis
    - Cross-VPC relationship highlighting
    - Per-security-group detailed views

- Visualization Options
    - Interactive visualization using Plotly
    - Static visualization using Matplotlib
    - VPC boundary visualization
    - Configurable node and edge styling
